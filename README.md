## Installation
Clone this repository and ensure NodeJS, npm (or any npm alternative such as yarn or bun) and PostgreSQL 16.4 is installed.

```shell
# login to postgres terminal and run the following commands:
\i /path/to/repo/schema.sql
\i /path/to/repo/seeds.sql
# npm
npm -i
# bun
bun -i
```
  

## Usage
After installation of the application run either of the following commands within the terminal of the cloned repo: 

```shell
# npm
npm run start
# bun
bun run start
```
